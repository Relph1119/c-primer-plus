# 习题2.3

&emsp;&emsp;编写一个程序把你的年龄转换成天数，并显示这两个值。这里不用考虑闰年的问题。

**解答：**
```c
#include
<stdio.h>
int main(void){
    int age_years, age_days;
    // 设置年龄
    age_years = 32;
    // 计算天数
    age_days = 365 * age_years;
    // 打印并显示年龄和天数
    printf("The age of %d years is %d days.\n", age_years, age_days);
    return 0;
}
```
