# 习题5.7

&emsp;&emsp;编写一个程序，提示用户输入一个`double`类型的数，并打印该数的立方值。自己设计一个函数计算并打印立方值。`main()`函数要把用户输入的值传递给函数。

**解答：**
代码位置：`exercises/ch05/ex07.c`

```c
#include <stdio.h>

void print_cube(double x);

int main(void) {
    double num;

    // 提示用户输入一个double类型的数
    printf("Enter a double value:");
    scanf("%lf", &num);
    // 调用函数，打印该数的立方值
    print_cube(num);

    return 0;
}

void print_cube(double x) {
    // 计算并打印x的立方值
    printf("The cube of %g is %g.\n", x, x * x * x);
}
```
